# 速问速答 Q&A

## 死亡后会掉落物品吗？
::: info 答
> 会的，死亡后会掉落您的物品。
:::

~~## 服务器是否支持跨版本？~~
::: info 答
~~> 是的，服务器支持跨版本登录。~~
:::

## 地皮/领地 管理
::: warning 关于圈地
玩家可以使用 `/dom` 命令进行领地操作。目前每名玩家只有一个 **64×64** 的领地。后续会根据服务器情况开放其他获取方式。
详见[领地系统](/server/rules)
:::



## 可以公布种子吗？

::: info 答
> 可以，服务器种子为：`1316390934843701335`
:::

## 支持挂机假人吗？
::: info 答
> 不支持挂机假人，但玩家可以通过小号进行挂机。
:::

## 被熊攻击了怎么办？
::: info 答
> 如果遇到熊攻击导致问题，可以主动联系管理员，我们可以为您提供回档服务。
:::

## 有刷铁机 / 刷线 / 刷地毯机吗？
::: info 答
> 服务器没有主城，鼓励玩家自主互动，因此没有预设自动刷机。
:::


## 服务器的配置怎么样？
::: info 答
> 服务器配置：`Intel i7-14700KF` 独立机。
:::

## 服务器会晚上关服吗？
::: info 答
> 全天候在线，随时可以进入。
:::


## 平均在线人数是多少？
::: info 答
> 平均在线人数取决于节假日和玩家活跃度。
:::
