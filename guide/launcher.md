# 启动器下载与选择指南

为了顺利进入服务器游玩，你需要使用 **Minecraft 启动器** 来启动游戏。  
以下将介绍几种常见且稳定的启动器，并说明它们的适用人群，方便你根据自身情况进行选择。

---

## 启动器简介
::: info 启动器用于
- 下载并启动 Minecraft
- 管理游戏版本
- 安装模组、整合包
- 配置 Java 与运行参数
:::

`本服务器 **不强制要求使用特定启动器**，只要能够正常进入游戏即可。`

---

## 推荐启动器列表

### PCL（Plain Craft Launcher）

**适合人群：**
- 新手玩家
- 希望界面友好、操作简单的用户

::: tip 适合
**第一次接触模组服或服务器的玩家**
:::

`爱发电下载:` [PCL Launcher](https://afdian.com/p/0164034c016c11ebafcb52540025c377)

---

### HMCL（Hello Minecraft! Launcher）

**适合人群：**
- 有一定 Minecraft 使用经验的玩家
- 希望在易用性与自由度之间取得平衡的用户

::: tip 适合
**大多数普通玩家，推荐使用**
:::

`官网下载:` [HMCL Launcher](https://hmcl.huangyuhui.net/download/)

---

### Prism Launcher

**适合人群：**
- 进阶玩家
- 熟悉模组管理与实例配置的用户

::: warning 注意事项
- 界面相对简洁，学习成本略高
- 不完全适合纯新手
:::

::: tip 适合
**追求稳定性与高度自定义的玩家**
:::

`官网下载:` [Prism Launcher](https://prismlauncher.org/download/windows/)

---

## 启动器选择建议

| 玩家类型 | 推荐启动器 |
|--------|------------|
| 完全新手 | PCL |
| 普通玩家 | HMCL |
| 进阶 / 模组玩家 | Prism Launcher |

> 如果你不确定选哪个，**直接使用 PCL 通常不会出错**。

---

## 版本与模组说明

- 请确保使用 **服务器指定的游戏版本**
- 若服务器需要模组或整合包，请按照服务器说明进行安装
- 启动失败时，可尝试：
    - 更换 Java 版本
    - 检查模组是否齐全
    - 重新创建实例

---

## 常见问题

- **启动器之间可以互换吗？**
::: info 答
  可以，但不同启动器的实例与配置通常不互通。
:::

- **一个电脑可以安装多个启动器吗？**
::: info 答
可以，互不影响。
:::
